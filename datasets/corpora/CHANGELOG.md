# Corpora Changelog

This file tracks additions, removals, and edits to the site manifests in
`datasets/corpora/`.

## Removal policy

If the operator of a site listed in any corpus requests removal, follow
[`rules/anti-clone-policy.md`](../../rules/anti-clone-policy.md) §7:

1. Remove the site entry from the relevant corpus JSON.
2. Delete every screenshot under `datasets/captures/` for that host.
3. Delete every observation under `datasets/observations/` for that host.
4. Append a dated entry to this file.
