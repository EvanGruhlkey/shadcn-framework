# Launchframe

AI-ready Next.js starter for turning any site into your SaaS landing page.

The CLI copies a bundled template (Next.js, shadcn/ui, agent rules, and the `/launchframe` slash command), runs `npm install`, and you’re ready.

```bash
npx launchframe@0.5.2 my-app
cd my-app
```

Then in your AI agent:

```
/launchframe https://example.com "Your SaaS idea"
```
